## Run Locally

1- Clone the project

```bash
  git clone https://github.com/MohamadSalamaMouse/GradTeam
```

2- to run project after clone

```bash
  1 - run composer install
  2 - change .env.example to .env
  3 - run php artisan key:generate
  4 - configure .env
```
